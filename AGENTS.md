## Package Instructions

This repository is the Spatie Laravel Data Skir generator.

- npm package name: `skir-laravel-data-generator`.
- It should generate Spatie Laravel Data objects for Skir structs.
- Generated PHP should depend on `spatie/laravel-data` for DTO creation and validation.
- Generated PHP should depend on `php-skir/runtime` for wire-format behavior.
- Keep server routing and client HTTP behavior out of this package.
